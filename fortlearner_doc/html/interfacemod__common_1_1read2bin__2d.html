A subroutine to read a CSV file and dump it as a binary file.


NOTICE: Can't handle multiple data types at the same time.
Use 'read_bin' subroutine to read the dumped binary file.
The type of the output data (32bit or 64bit) is determined by the type of the argument.
Subroutines of other data types (read2bin_2d_64bit) are stored in './common/include/read2bin/'.

Returns
returns none
Parameters
input_file_nameinput file name
output_file_nameoutput file name
n_rowsnumber of lines in the input file(header not included). 'n_rows' should be less than or equal to the number of lines in the input file.
n_colsnumber of columns in the input file. 'n_cols' must be equal to the number of lines in the input file.
skip_headerwhether to skip the header. If .true.,
input_dtypeinput data type, 'r' or 'i' ('real' or 'integer')
output_dtypeoutput data type, 'r' or 'i' ('real' or 'integer')
